Integration setup guide
Step-by-step instructions for connecting live data
How it works: Enter your API credentials in Settings (⚙️). The dashboard saves them only in your browser's local storage — never sent to any server.
Fb
Facebook & Instagram (Meta Graph API)
1
Create a Meta Developer App
Go to developers.facebook.com/apps → "Create App" → Choose "Business". Link your Facebook Page and Instagram Business account.
2
Add the Pages API & Instagram Graph API products
In your app dashboard click "Add Product" and add Facebook Login and Instagram Graph API.
3
Get a long-lived Page Access Token
Use the Graph API Explorer to generate a token with pages_read_engagement, instagram_basic, instagram_manage_insights permissions. Exchange for a long-lived token (60-day expiry).
4
Get your Page ID & Instagram Account ID
Call GET /me/accounts in Graph Explorer. For Instagram call GET /{page-id}?fields=instagram_business_account. Paste both IDs into Settings.
YT
YouTube (YouTube Data API v3)
1
Create a Google Cloud project
Go to console.cloud.google.com → New Project → Enable YouTube Data API v3 from the API Library.
2
Create an API key
Credentials → Create Credentials → API Key. Restrict it to "YouTube Data API v3". Paste into Settings → YouTube API Key.
3
Find your Channel ID
On YouTube, go to your channel → About → Share → Copy Channel ID. Paste into Settings.
TT
TikTok (Display API)
1
Apply for TikTok Developer access
Go to developers.tiktok.com → Create App → Select "Display API". Note: TikTok requires app review.
2
Get your access token
After approval, use OAuth 2.0 to get a user access token with scopes user.info.basic and video.list. Paste into Settings.
in
LinkedIn (Marketing API)
1
Create a LinkedIn app
Go to linkedin.com/developers/apps → Create App. Associate it with your company LinkedIn Page.
2
Request Marketing Developer Platform
In Products, request Marketing Developer Platform. Approval may take a few days.
3
Get an access token
Use OAuth 2.0 with scopes r_organization_social and rw_organization_admin. Paste token and Organization URN into Settings.
G
Google Business Profile API
1
Enable the API
Google Cloud Console → API Library → search "Business Profile Performance API" → Enable.
2
Create OAuth 2.0 credentials & authorize
Use OAuth Playground with scope https://www.googleapis.com/auth/business.manage. Paste your refresh token and location name into Settings.
GA
Google Analytics 4
1
Enable GA4 Data API in Google Cloud
API Library → Enable Google Analytics Data API.
2
Create a Measurement Protocol API Secret
GA4 Admin → Data Streams → Measurement Protocol → Create secret. Also copy your Property ID (Admin → Property Settings).
3
Paste into Settings
Enter your GA4 Property ID (e.g. 123456789) and the API secret into the GA4 section in Settings.
Settings & credentials
